# KumaNORETA

KumaNORETA（くまのれた）は、熊本県内のGTFS-JP / GTFS-Realtimeデータを活用し、現在乗車中のバスを推定して降車予定時刻を表示するWebアプリです。

名前は「熊本」+「乗れた」+「NOR-ETA」から来ています。

## Concept

Googleマップなどの経路検索は「乗る前」の案内に強い一方で、KumaNORETAは「乗った後」の支援に特化します。

- 今、自分がどのバスに乗っているかを推定する
- 降りたい停留所を選ぶ
- 到着予想時刻を表示する
- 家族への迎車共有に使える情報を出す
