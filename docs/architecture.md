# アーキテクチャ設計メモ

## 目的

KumaNORETAは、スマートフォンGPS、GTFS-JP、GTFS-Realtimeを組み合わせて、利用者が現在乗車中のバスと降車停留所へのETAを推定する。

## 主要コンポーネント

- GPS取得: ブラウザGeolocation APIから現在位置を取得する。
- GTFS-RT取得: VehiclePositions / TripUpdatesを取得し、車両位置と遅延情報を正規化する。
- GTFS-JP読込: 停留所、路線、便、停車順を読み込む。
- 乗車中バス推定: GPS位置、車両位置、経路、進行方向から候補をスコアリングする。
- ETA計算: GTFS-RTと静的時刻表をもとに降車停留所到着時刻を推定する。
- UI表示: 推定候補、信頼度、降車停留所、ETAを表示する。
